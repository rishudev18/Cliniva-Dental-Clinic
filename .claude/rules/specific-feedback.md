---
name: specific-feedback
description: Be specific when comparing screenshots and identifying mismatches
type: rule
---

When comparing screenshots, be specific about what's wrong.

Use concrete measurements and clear descriptions when identifying differences between your rendered output and the reference image. For example:
- "heading is 32px but reference shows ~24px"
- "gap between cards is 16px but should be 24px"
- "color is #FF0000 but reference uses #E63946"

Vague feedback like "looks off" is not acceptable. Always cite measurements or specific visual properties that differ.
