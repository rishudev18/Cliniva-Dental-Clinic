# Website Design Recreation

This project recreates website designs from reference images (screenshots) using HTML and Tailwind CSS. The goal is to match the reference design precisely within ~2–3px of accuracy.

## Rules

- [[no-feature-additions]] — Do not add features, sections, or content not present in the reference image
- [[exact-design-match]] — Match the reference exactly; do not "improve" the design
- [[use-provided-tokens]] — Use CSS classes or style tokens provided by the user verbatim
- [[inline-tailwind]] — Keep code clean but don't over-abstract; inline Tailwind classes are fine
- [[specific-feedback]] — Be specific when comparing screenshots and identifying mismatches
- [[iterate-multiple-rounds]] — Always do at least 2 comparison rounds; do not stop after one pass
